# 🚀 Render Deployment Test

A minimal Flask application created to verify deployment on Render using Gunicorn.

## Tech Stack

- Python
- Flask
- Gunicorn
- Render

## Run Locally

```bash
pip install -r requirements.txt
python app.py
```

Visit:

```
http://localhost:10000
```

The application returns:

```
WhatsApp Bot Running 🚀
```

## Purpose

- Test Render deployment
- Verify Gunicorn configuration
- Validate Flask application hosting
